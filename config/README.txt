1- Change the name of the database_infos-template-.json file to database_infos.json.
2- Create a google cloud bucket and mssql database then give their infos to database_infos.json.
3- Create a dir in bucket with name "photos".
4- Upload a default profile photo in this dir with name "default_pp".
5- Download your bucket key json file in this config dir.
6- Create a google cloud database with mssql and give database_infos.json its infos.
7- You must create your db and tables and change querys in this backend codes.
8- You are ready!
